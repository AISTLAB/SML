Welcome to SML Pages.

This project maintain a Virtual Machine mainly using C Language, Our purpose are creating a minimal Virtual Machine for:

  • Teaching students

    1. How cpu works
    2. How assembly language turns into machine language
    3. How high level language turns into assembly language
    4. Why algorithm is so important, and How to write effective code step by step

  • Using for production

    1. Finally we want portable the code to the Mainly embedded platform, 89C51, Arduino, STM32F103, Raspberry Pi, Web Service,etc, so we can realize "programming once, run every where".

    2. The benifit is the newcomer can migrate their experience discard the complex detail of the hardware.
